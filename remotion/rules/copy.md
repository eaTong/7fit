# 文案规范（remotion）

> 阶段：文案 / 脚本创意（copy / voiceover / 钩子设计）
> 适用场景：用户要"出文案"、做配音旁白、生成语音 → 字幕前的文字稿
> 来源：用户硬约束 + 七练品牌调性
> 状态：✅ 生效
> 上下游：上游 = `resources/docs/`（产品/品牌事实）+ `video-types.md`（确定类型 A/B/C）；下游 = `subtitle.md`（文案→音频→字幕） + `storyboard.md`（文案→分镜）

---

## 1. 文案工作流

### 1.1 触发条件

用户说"出文案 / 写旁白 / 写脚本 / 写文案稿"时，按本规范执行。

### 1.2 流程

```
resources/docs/  (产品事实 + 品牌文案事实)
       ↓
1. 选题定位：明确这条文案给谁看、解决什么问题
       ↓
2. 钩子设计：3 秒抓眼球（见第 3 节）
       ↓
3. 口语化撰写：按"对话感"写（见第 2 节）
       ↓
4. 违禁词自检（见第 4 节）
       ↓
5. 落盘到 resources/docs/copy/<视频主题>.md
       ↓
（可选）TTS 生成音频 → resources/audios/<主题>.mp3
       ↓
下游：mmx 识别音频 → subtitles.json（subtitle.md）
```

### 1.3 输出位置

文案稿放在 `resources/docs/copy/`：

```
resources/docs/
├── SUMMARY.md
├── prd.md
├── opc-niche.md
└── copy/                   # 文案稿
    ├── workout_intro.md    # 一份视频一个 .md
    ├── weekly_review.md
    └── ...
```

每份文案稿包含：选题 / 目标受众 / 钩子 / 完整旁白 / 违禁词自检 / 备注。

## 2. 口语化（最核心）

**文案必须像人说话，不像品牌宣传**。七练主理人是"和你一样在练的同侪"——文案要传达这种人格。

### 2.1 ✅ / ❌ 对比

| ❌ 广告腔 | ✅ 口语化 |
|---|---|
| 引领智能健身新时代 | 我跟你说，健身这事其实没你想的那么难 |
| 颠覆传统，AI 助力精准训练 | 这半年我训练一直卡在一个数据上，突然意识到问题 |
| 立即体验，开启智能健身之旅 | 试试这个，反正不花你 30 秒 |
| 我们致力于为... | 我自己做了一个东西 |
| 突破极限 | 终于把重量推上去了 |
| 享受 AI 红利 | 用了之后才发现... |

### 2.2 口语化 6 条军规

1. **用第一人称（"我")和第二人称（"你")**——少用"我们/本产品/七练"
2. **短句优先**——一句话不超过 20 字，3-5 字短句穿插节奏
3. **允许语气词**——"哎"、"其实"、"说真的"、"反正"、"我跟你说"
4. **允许口语停顿**——"……" "——" 用作语气节奏
5. **允许不完整句**——对话中常说半句话就转话题（但正文里要能看懂）
6. **不堆砌形容词**——"智能/高效/精准/极致"这类抽象词一个文案里 ≤ 1 个

### 2.3 推荐表达模板

```
开场钩子：[一个反常识/痛点共鸣/具体场景] + 1-2 句口语
陈述段：我自己 [做某事] + [发现/经历] + [转折]
论证段：举个例子 / 你有没有发现 [共鸣] + [数据/对比]
收尾：所以我做了 [产品/动作] + 软 CTA（不强制购买）
```

## 3. 前 3 秒钩子

视频/口播/封面文案，**前 3 秒必须有钩子**。3 秒抓不住人就划走了。

### 3.1 钩子 5 种类型

| 类型 | 公式 | 适用 | 示例 |
|---|---|---|---|
| **反常识** | "其实……" + 反转认知 | 新知类 | 其实健身这事儿，记住数据比做对动作更值钱 |
| **痛点共鸣** | "你是不是也……" | 方法/工具类 | 你是不是也练了大半年，连自己上周练了多少都不知道 |
| **数字冲击** | "[反常数字] + [震撼结论]" | 数据/结果类 | 30 秒 vs 3 秒，差距 10 倍 |
| **身份代入** | "[身份] + [场景]" | 同侪/经验类 | 练了 2 年，每次称重才发现自己早该调整计划 |
| **悬念/提问** | "[问题] + [为什么]" | 教程/揭秘类 | 为什么你练胸总是没感觉？我发现一个被忽略的事 |

### 3.2 钩子的硬要求

- **一句话长度** ≤ 25 字（中文）
- **第 1 个字就要抓人**——禁止"大家好，我是…""今天给大家分享…"这种慢热开头
- **不绕弯**——不堆背景、不自我介绍、不寒暄
- **最好有数字/对比/反差**——具体的东西比抽象的更容易抓眼球
- **必须与视频主题强相关**——不要"标题党"

### 3.3 ❌ 禁用开头

- "大家好，我是…"
- "今天给大家分享…"
- "在开始之前，我想先说…"
- "你是否曾经有过这样的困扰…"
- "在这个快节奏的时代…"
- 任何寒暄、铺垫、官方感开场

## 4. 违禁词规则（抖音 / 小红书）

> **来源**：抖音《社区自律公约》、《抖音社区财经/法律行业公约》、《广告法》第九条；小红书《社区公约 2.0》（2026-01）、直播笔记违禁词与违规问题手册（2025 版）。
> **共同基线**：两平台都执行《广告法》第九条绝对化用语禁令 + 价值观/法律底线（不涉政、不涉黄、不涉暴、不涉未成年人不良内容）。
> **更新日期**：2026-06-03

### 4.1 绝对化用语（极限词）—— **两平台共同最高频处罚**

完整禁用词：

- **"最"系**：最、最佳、最具、最爱、最赚、最优、最优秀、最好、最大、最大程度、最高、最高级、最高档、最奢侈、最低、最低级、最低价、最底、最便宜、时尚最低价、最流行、最受欢迎、最时尚、最聚拢、最符合、最舒适、最先、最先进、最先享受、最后、最后一波、最新、最新科技、最新科学
- **"一"系**：第一、中国第一、全网第一、销量第一、排名第一、唯一、第一品牌、NO.1、TOP.1、独一无二、全国第一、一流、一天、仅此一次（一款）、最后一波、全国 X 大品牌之一
- **"级/极"系**：国家级（相关单位颁发的除外）、国家级产品、全球级、宇宙级、世界级、顶级（顶尖/尖端）、顶级工艺、顶级享受、极品、极佳（绝佳/绝对）、终极、极致
- **"首/家/国"系**：首个、首选、独家、独家配方、全国首发、首款、全国销量冠军、国家级产品、国家（国家免检）、国家领导人、填补国内空白
- **"品牌"系**：王牌、领袖品牌、世界领先、领导者、缔造者、创领品牌、领先上市、至尊、巅峰、领袖、之王、王者、冠军

替换策略：

| ❌ 禁用 | ✅ 替换 |
|---|---|
| 最好的 | 很多用户推荐的 / 我用下来觉得很 OK 的 |
| 销量第一 | 卖得不错 / 我身边不少朋友在用 |
| 国家级 | 高品质 / 我自己挑了很久的 |
| 顶级 | 质量很好 / 真材实料 |
| 行业领先 | 做得比较早的 / 经验丰富的 |

### 4.2 承诺与保证

禁用词：无效退款、不满意退款、零风险、特效、强效、速效、保证、彻底解决、1 天见效、100%、包过、必过、一次通过、押题命中率百分百、就业保障、百分百高薪就业、过目不忘、稳赚不赔、保本、升值回报、投资理财零风险。

替换策略：把"承诺"改成"描述"，把"保证"改成"反馈"。

| ❌ 禁用 | ✅ 替换 |
|---|---|
| 1 天见效 | 我用下来 X 天开始感觉到 |
| 100% 有效 | 对我来说效果挺明显的 |
| 包过 | 系统学习 + 模拟练习 |
| 稳赚 | 长期看收益不错（但有风险） |

### 4.3 医疗暗示 / 功效夸大（七练健身赛道重点）

> ⚠️ 健身 ≠ 医疗，**普通食品/普通商品/健身训练方法**不能宣称医疗功效。

禁用词：

- **医疗术语**：治愈、根治、疗效、治疗、消炎、止痛、降三高、降血糖、抗癌、防癌、预防疾病、防治、活血化瘀、修复受损细胞、改善疾病
- **保健功效（普通食品禁）**：增强免疫力、调节内分泌、延缓衰老、排毒、养颜、祛斑、祛痘、防脱发、生发、助眠、增肥、瘦身、燃脂、溶脂、塑形、暴汗、暴瘦
- **美容/化妆类违规**：速效美白、一洗白、7 天祛斑、3 天淡纹
- **减肥/瘦身（抖音/小红书重灾区）**：
  - 7 天瘦 X 斤、X 天暴瘦
  - 不瘦退款、签约减肥、瘦身速成
  - 躺着瘦、躺瘦神针、碳水脸
  - 液断、生酮、节食、催吐、暴汗服
  - 任何涉及司美格鲁肽 / 替尔泊肽 / 减肥针的内容
- **健身/训练效果承诺**：
  - X 天练出马甲线、X 天拥有 X 腹肌
  - 一周 X 公斤肌肉
  - 包教包会学会深蹲/卧推/硬拉

替换策略：用"感受/数据/对比"代替"效果承诺"。

| ❌ 禁用 | ✅ 替换 |
|---|---|
| 治愈失眠 | 我自己用下来睡得踏实了点 |
| 7 天瘦 10 斤 | 我用 X 个月慢慢减下来 |
| 暴汗燃脂 | 出汗挺多的 |
| X 天练出马甲线 | 练了 X 周，体脂降了一些 |
| 治疗颈椎病 | 肩颈没那么僵了（仅描述感受，非疗效） |

### 4.4 权威性 / 涉政 / 国家机关

禁用词：国家 XX 领导人推荐、国家 XX 机关推荐、国家 XX 机关专供、特供、专供、人民币图样（央行批准的除外）、老字号（特定情况）、中国驰名商标、质量免检、无需国家质量检测、免抽检、填补国内空白。

替换策略：用品牌历史、用户口碑、第三方测评代替"国家背书"。

| ❌ 禁用 | ✅ 替换 |
|---|---|
| 国家 XX 机关专供 | 自家用了多年的 / 我们一直选这个 |
| 中国驰名商标 | 老牌子 / 圈里口碑不错 |

### 4.5 迷信 / 玄学

禁用词：增强第六感、化解小人、增加事业运、招财进宝、健康富贵、提升运气、护身、平衡正负能量、旺人、旺财、旺宅、镇宅、消灾、挡灾、助吉避凶、转富招福、冥器、旺夫、旺子、带来好运气、逢凶化吉、避凶、辟邪、防小人、算命、算八字、运势。

替换策略：七练本身是产品/工具，与此无关，**直接删掉**即可。

### 4.6 金融 / 投资 / 虚拟货币

> 抖音 2025-12 发布《社区财经行业公约（试行）》；小红书 2026-06 启动金融领域认证专项治理。

禁用词：收益率、投资回报、保底收益、返本销售、无风险回报、快速回报、高收益回报、升值、众筹、内部消息、稳赚、躺赚、暴富、年入百万、财富自由。

涉及虚拟货币/数字资产一律禁：区块链、数字资产、虚拟货币、炒币、挖矿、ICO、IEO、DEX、合约交易。

### 4.7 诱导互动 / 导流 / 引流

> 小红书 2025-2026 重拳治理。

禁用行为：

- **联系方式**：微信号、手机号、QQ、邮箱、二维码、网址链接
- **第三方平台**：淘宝、京东、拼多多、闲鱼、转转（除非平台官方合作）
- **引导性词汇**：扫码、加我、点击获取、点击有惊喜、点击试穿、点击转身、点击翻转、领取奖品、戳这里、看主页、私我
- **诱导互动**：拍了的姐妹在屏幕上打已拍、点关注优先发货、评论区扣 1 领取
- **虚假限时**：限时必须 X 天 X 时 X 分、最后一波、再不抢就没了、错过就没机会了、万人疯抢、卖/抢疯了、不会更便宜了

替换策略：把"外部引流"改为"站内互动"；把"虚假紧迫"改为"实际节奏"。

| ❌ 禁用 | ✅ 替换 |
|---|---|
| 加微信看更多 | 主页置顶有完整版 |
| 扫码进群 | 评论区互动，我整理好会发出来 |
| 限时最后一天 | 本周活动，本周日结束 |
| 万人疯抢 | 很多人在用 |

### 4.8 贬低拉踩 / 制造对立

> 小红书《社区公约 2.0》（2026-01）专门把"反对制造对立"列入倡导。

禁用行为：

- 拉踩同行产品/品牌
- 标题党（"千万不要买"、"买了后悔"——种草文伪装避雷贴）

### 4.9 虚假人设 / 编造经历

> 小红书重拳治理"虚假人设"。

禁用行为：

- 假装自己是素人/学生/妈妈/医生（实际不是）
- 伪造使用体验（"我女儿/当妈才知道"等拟人视角）
- 借用他人图片/视频（盗用素材）

### 4.10 涉未成年人 / 涉黄 / 涉暴

> 两平台零容忍。

- 任何涉未成年人擦边（穿着暴露、性感姿势、未成年人出镜带货等）
- 任何色情、低俗、性暗示

### 4.11 行业专项（七练健身相关）

- **滋补保健**：禁止宣传壮阳、丰胸、补肾等功效；不得暗示产品可替代药品

## 5. 内容风格：生活化对话（不要广告感）

### 5.1 调性目标

- **像朋友聊健身**，不像品牌发新闻稿
- **承认不完美**——"我也会偷懒/也会练砸/也会忘记录"
- **带具体细节**——具体时间、地点、重量、次数、画面
- **敢用第一人称吐槽**——"说实话这个我做之前也犹豫"

### 5.2 ❌ 广告感 6 个特征

| 特征 | ❌ 例子 | ✅ 改写 |
|---|---|---|
| 频繁使用"我们/本产品" | "我们提供 AI 私教" | "我做了个东西，专门解决这个" |
| 堆砌优势词 | "智能高效精准" | "3 秒记完，比以前快 10 倍" |
| 强制 CTA | "立即扫码加入" | "试试看，反正不花钱" |
| 完美主义口吻 | "完美记录每一次训练" | "记录这事儿，卡 1 次不算事儿" |
| 强调"颠覆/革命" | "颠覆传统训练方式" | "以前我每周手动记 Excel，现在懒得记也能有数据" |
| 假装客观 | "经测试，效率提升 300%" | "我自己用下来，3 秒就够了" |

### 5.3 生活化对话的 5 种开场

- **场景开场**："周六早上去健身房..."
- **痛点开场**："你有没有练完就忘..."
- **结果开场**："上周我终于把卧推推到了 80..."
- **矛盾开场**："很多人觉得练得勤就行，其实..."
