# C-1 · 分镜（Storyboard）v3

> **视频**：c01_pm_fitness_log
> **类型**：C · 七练解码（身份反差 + 造轮子混合）
> **呈现**：口播为主（右下角正方形） + 信息密度拉满（每段 ≥ 3 信息元件）

## 口播布局

口播视频位于画面**右下角，280×280 正方形容器**，`object-fit: cover`，贯穿全片。

```
┌──────────────────────────────────┐
│ ╔══ HUD $ 7fit decode ═══╗     │  HUD头栏80px
│ ║  status: 解码中         ║     │  始终显示
│ ╚══════════════════════════╝     │
│                                  │
│ ┌─── 信息元件区 ───────────┐    │
│ │                            │    │  主区域占画面65%
│ │  卡片/数据/文字/高亮        │    │  填充整个上部+左部
│ │  每段≥3个信息元件            │    │
│ │                            │    │
│ └────────────────────────────┘    │
│                    ┌──────┐       │
│     右上辅助信息区  │ 口播  │       │  口播280×280
│                    │ 280² │       │  右下角
│                    └──────┘       │
│ ▒▒ 底部状态栏 ▒▒▒▒▒▒           │  48px
└──────────────────────────────────┘
```
