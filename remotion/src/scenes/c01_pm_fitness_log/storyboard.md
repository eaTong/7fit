## 口播分段方案

### 为什么改成分段

| 单段连续自拍 | 分段自拍 |
|---|---|
| 一次录 93s 容易累场、表情僵 | 6 段，每段 10-15s 容易保持状态 |
| 表情和段落主题不匹配（皮质醇段和 PM 段表情差不多）| 每段表情对应场景：皮质醇→严肃、PM→理性、PM翻译→推眼镜、CTA→轻松 |
| 只能用一种机位/光线 | 每段可微调机位（特写 / 半身近景 / 双手比划）|
| 后期调音很难（一句卡壳要重录整段）| 卡壳只影响一段 |

## 自拍拍摄清单

> 给执行拍摄的人看的清单。

### 准备

- **设备**：iPhone / 任何手机或相机
- **光线**：窗户自然光 + 主光（避免顶光造成眼影）| 背景：干净不杂乱
- **机位**：水平视线，建议架在三脚架或书堆上（避免手持抖动）
- **录音**：手机自带麦克风，**距离嘴 30-50cm**，安静环境

### 拍摄参数建议

- **分辨率**：1080p 30fps 竖屏 9:16
- **每段独立成片**：每段录 2-3 条，挑最好的
- **格式**：横屏/竖屏均可，最终 `object-fit: cover` 裁剪成正方形
- **如果横屏录**：录完整上半身，方便后期裁剪
- **如果竖屏录**：录脸 + 手部动作，方便直接剪

### 各段拍摄要点

| 段 | 段名 | 要点 |
|---|---|---|
| TA0 | 钩子 | 节奏紧、字字清晰、眼神有冲击力 |
| TA1 | 皮质醇 | 略慢、回忆式、可以闭眼 0.3s 表示"当时状态" |
| TA2 | 反思 | 自问自答、可以点头 / 摇头配合 |
| TA3 | PM 翻译 | 推眼镜 / 手指划线 / "等价于"用手比划"=" |
| TA4 | 小程序 | 拿起手机道具展示、或用手势"说话" |
| TA5 | 三场景 | 眼神左右看 3 次（"切场景"），语速稍快 |
| TA6 | 金句 | 节奏加重、眼神坚定、"30 秒"看镜头、"3 秒"再点一下头 |
| TA7 | CTA | 微笑、轻松姿态、引导式手部动作 |

## 关键剪辑规范

### 口播框显示规则

- **正方形 280×280**（`object-fit: cover`）
- **位置**：右下角（X = 1920-280-32, Y = 1080-280-80）
- **入场**：每段口播入场时 0.3s spring 弹入（scale 0.8→1）
- **退场**：转场前 0.3s fade out
- **过渡转场**：wipe-h/wipe-v/glitch 在口播框**之外**的主要画面上演，不让转场吃掉口播内容

### 转场与口播的时序

```
┌─ TA_n 主体内容（10-15s）─┐──转场（0.3-0.5s）──┌─ TA_n+1 主体内容 ──┐
   口播框入场 0.3s       主体内容           口播框退场 + 主体转场     口播框入场
   ↑                                          ↑                              ↑
   这段时间口播全程显示               主体内容做转场（数字弹入、卡   新段口播开始
                                        片切换、glitch）时口播暂        闪现
                                        时退场
```

> **关键**：**转场期间口播可以短暂退场**（0.3-0.5s），让转场效果更醒目。但每段主体期间口播要持续可见。
