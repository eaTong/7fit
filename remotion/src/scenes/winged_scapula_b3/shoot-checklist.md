# 拍摄清单：翼状肩胛自测 + 改善：4 个动作

> ⚠️ **拍摄当天打开这个文件**。素材规格见 [assets.md §1](assets.md#1-全部资产单表按-排序) 「拍摄要求」列。
> 总素材数：7 段视频。

---

## 1. 通用要求（所有 7 段都遵守）

- [ ] **着装**：穿贴身运动衣 / **必须露出肩胛骨**（不能宽松遮挡）
- [ ] **背景**：浅色墙（健身房一角 / 居家干净墙角）/ 避免杂物
- [ ] **设备**：三脚架固定 / 或找朋友帮拍
- [ ] **画质**：1080p / 30fps / 4K 更好 / 横平竖直
- [ ] **收音**：iPhone 单独录旁白（不用视频自带 mic）
- [ ] **试拍**：拍前先 1 段测试，检查焦点、亮度、肩胛骨可见性
- [ ] **录前**：深呼吸 3 次，让嗓子放松
- [ ] **录中**：保持坐姿（旁白时）/ 站姿（演示时）嘴离话筒/手机恒定距离

---

## 2. 各段特别提醒

> 每条对应 assets.md §1 中的 #。**只列重点**，完整规格见 assets.md。

- [ ] **#1（钩子对比）**：背对镜头 + 慢转 30°；先拍正常站姿 3s，再拍翼状站姿 3s（不刻意做，但用嘴示意"突出"位置）
- [ ] **#2（镜子自测 1）**：**需要朋友**；朋友站镜子侧前方拍；自己背对镜子、肩完全放松、不要挺胸
- [ ] **#3（推墙测试）**：**需要朋友**；朋友站斜后方拍；推墙保持 5-8 秒（看肩胛骨变化），收回 2 秒
- [ ] **#4（壁虎推墙）**：面对墙、**脚尖离墙 15-30cm**；前臂和手掌贴墙 / 大小臂 90°；**全程肩胛骨下沉**（不耸肩）；慢速演示
- [ ] **#5（俯卧撑+前伸）**：标准姿或跪姿（新手）；核心是"前伸"那一下（让上背呈"凹陷"），然后收回中立
- [ ] **#6（YTWL）**：**背对镜头拍**（唯一正确机位）；手握轻哑铃或矿泉水瓶；Y→T→W→L 顺序；每字母停 2 秒
- [ ] **#7（弹力带后收）**：背对镜头；**重点是肩胛骨向中间夹**，不是手臂拉开的距离
