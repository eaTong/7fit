## 5. 排期例外规则

### 跳过排期的情况

- 用户明确说"这个星期不发"
- 内容没准备好（不是每周都必须发 3 条）
- 重大事件（节日、个人原因）→ 整周跳过

### 补发规则

- 跳过 1 周 = 下周补齐
- 跳过 2 周 = 不补，标记为"停更期"
- 跳过 3 周+ = 复盘节奏

### 提前排期

- 重大产品发布（v2.0 / 新功能）→ 可在 M1 计划外加 1-2 条 C 类
- 热点事件（健身圈新闻）→ 可临时插入 B 类
