## 1. 选题拆解（用户原话 → 3 个子角度）

**用户原话**：*"A 类，OPC 相关的，介绍一下我当前做的项目：用 vibe coding 做小程序，并 vibecoding 自动剪辑视频"*

| 子角度 | 主体 | 钩子方向 | 时长占比 |
|---|---|---|---|
| **A1-1 vibe coding 做七练小程序** | 工具栈（Cursor / Claude / mmx / TDesign）+ 过程（PRD → 搭 → 上线）+ 反思 | 身份代入 + 数字冲击 | 50% |

| **A1-3 反思 + 一人公司方法论** | 4 个 AI 工具替代 1 个团队 / 周 < 8h / 月投入 ≤1000 / 不融资不雇人 | 反常识 | 15% |

**主线选择**：
- **A1-1 + A1-2 = 主线**（90% 视频）—— 工具案例 + 流程
- **A1-3 = 收尾**（10%）—— 把"vibe coding"和"一人公司"挂上钩
