## 钩子布局规范（2×2 网格同屏 · B14 首发 · 以后复用）

> **规则**：钩子（前 3 秒）用 **2×2 网格** 同屏显示 **4 个动作**，动作对应"躺着推/靠着推/向上推/向前推"。

| 格子 | 动作 |
|---|---|
| 左上 | 躺着推 |
| 右上 | 靠着推 |
| 左下 | 向上推 |
| 右下 | 向前推 |

- 画布：1080×1920（竖屏）
- 每个格子：~480×480px，2×2 均分
- 间隔：8-12px 细分割线（纯色或白线)
- 无背景色，动作视频直接填满各格子（`object-fit: cover`）
- 字幕叠加在网格底部
