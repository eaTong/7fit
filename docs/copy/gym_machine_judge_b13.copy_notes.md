## 录音指引

> 用户选择"自录旁白"。下面是录音规范。

### 文件规格

| 项 | 要求 |
|---|---|
| **路径** | `../../../resources/audios/gym_machine_judge_b13.m4a` |
| 格式 | **M4A / AAC-LC**（iPhone 语音备忘录默认）|
| 采样率 | 44.1 kHz 或 48 kHz |
| 比特率 | **64 kbps+**（建议 128 kbps+）|
| 声道 | 单声道（mono）足够；立体声也接受 |
| 目标时长 | **~62.3 秒**（与 v6 文案匹配）|

### 录音设备

- **手机录音**（推荐）：iPhone 语音备忘录 / Android 录音机
- **Mac 内置**：QuickTime Player → 文件 → 新建音频录制

### 环境要求

- ✅ **安静环境**（关窗、关空调、避开马路）
- ✅ **话筒距嘴 10-20cm**（不要太近，否则喷麦）
- ❌ 不要在有回声的房间里录（浴室、空客厅）
- ❌ 不要有背景音乐（这步只录旁白，BGM 在分镜后单独加）

### 朗读节奏

- **总时长目标**：~62.3 秒（v6：3.4 字/秒 + 11 破折号 × 0.5s + 段 3 12.7s + 4 段间 × 0.7s）
- **语速**：自然偏快，**像在跟朋友解释**——不是念稿
- **破折号"——"** = **停顿 0.5 秒**（v4 新规则：比标点 0.3s 长）
- **"这里" 占位符** = 每个 0.5 秒停顿（让观众视觉上"看到"不同器械）
- **数字**：清楚念出（"十二次力竭" 而不是 "12 次力竭"）

### 录音技巧

- **录前**：深呼吸 3 次，让嗓子放松
- **录中**：保持坐姿，嘴离话筒恒定距离
- **段 1 重点**：每个"——"停顿 **0.5s**（v4 新规则：比标点 0.3s 长），让观众视觉上"看到"3 个不同器械（即使录音只是念字）
- **v4 关键**：12 破折号 × 0.5s = 段 1 多用 6s 时间；录音时要慢但稳，每个破折号都给足停顿
- **录错**：不用停——直接重念那句，自然过渡
- **录后**：听一遍，看是否需要重录某段
- **导出**：**保持 iPhone 语音备忘录原始格式（m4a / AAC）**——直接分享 → 存储到文件 → 选 m4a / 不转码
